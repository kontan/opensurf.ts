Speeded Up Robust Feature Demo

Abstract

This is a TypeScript(JavaScript) port of OpenSURFcs. You can execute SURF local feature detection on your Web browser. This demonstration is available on a Web browsers supports some advanced feature, like Canvas, Web Workers and Typed Array. Google Chrome is one of suitable browser for it.

Usage

Input a url of a image to the following text box:

Source Image URL:
(Caution: Cross domain image loading is not yet implemented. You can choose a image from only the following sample images.)
Sample Images: yamada.png akiko.png cat.png flower.png
When image loading finished, press the next button to execute detection.
